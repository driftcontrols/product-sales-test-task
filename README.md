# Sales Dashboard

Небольшой дашборд для отслеживания продаж по продуктам.

## Стек

- React 18 + TypeScript
- Redux Toolkit
- Ant Design 5
- Apache ECharts

## Запуск

```bash
npm install
npm run dev
```

Также можно работать внутри StackBlitz
https://stackblitz.com/edit/vitejs-vite-28hsg3vy?file=README.md

## Что умеет приложение

- Показывает список продуктов с сортировкой по продажам
- Визуализирует продажи в виде bar-chart
- Позволяет добавлять новые продукты через форму
